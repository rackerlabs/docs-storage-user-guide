.. _storage-intro:

======================================
Introducing Rackspace storage services
======================================
Services in this category include:

  * Cloud Block Storage, a block-level storage solution that enables you to
    expand the storage capacity of your cloud servers.

  * Cloud Files, a storage service that you can use to store an
    unlimited number of files and to publish and distribute content behind
    a content delivery network.

  * Rackspace CDN, a content delivery network service that you can
    use to accelerate your websites, web applications, and
    media delivery.
