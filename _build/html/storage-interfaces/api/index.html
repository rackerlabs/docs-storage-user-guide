If you write applications that are tightly integrated with the Rackspace cloud, you might find that clicking to choose options in the Cloud Control Panel and typing to issue commands with CLIs are not ideal for your purposes.

If you develop or distribute applications for your customers, or if you wish to heavily automate operation of your cloud infrastructure, you might need to interact with cloud services through one of the many available software development kits (SDKs) or by direct API access.

If you are responsible for administering systems that are used for cloud software development, you might want to investigate DevOps tools designed for this purpose.

To learn about how to use SDKs and APIs, start at the Rackspace Core Infrastructure User Guide.

For more information on using SDKs and APIs to interact with a specific cloud service, see:

  • Cloud Block Storage and SDKs and APIs
  • Cloud Files and SDKs and APIs
  • Rackspace CDN and SDKs and APIs
